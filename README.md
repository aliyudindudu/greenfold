# GreenFold 📦🍃

**Reimagining Waste into Wonder.**

GreenFold is a premium E-commerce catalog and landing page for handmade, upcycled cardboard crafts. Designed with an "Organic Industrial" aesthetic, it showcases how sustainable materials like corrugated cardboard can be transformed into high-end furniture, home decor, and toys.

![GreenFold Banner](https://images.unsplash.com/photo-1603530331624-9bdf01b7a2f5?q=80&w=2000&auto=format&fit=crop) *(Placeholder for project visual)*

## ✨ Features

- **Unfolding Animations:** Elements use 3D GSAP transforms to "unfold" into the viewport, mimicking the opening of a box.
- **Eco-Cycle Process:** An interactive scroll-triggered SVG path that draws itself as you learn about our sanitization and design process.
- **3D Craft Catalog:** Product cards with 3D tilt effects and "Recycled" stamp reveals on hover.
- **Texture-Rich UI:** Subtle paper grain textures and layered shadows for a tactile, physical feel.
- **Custom Commissions:** A bespoke request system for custom cardboard installations.

## 🚀 Tech Stack

- **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
- **Language:** TypeScript
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animation:** [GSAP](https://greensock.com/gsap/) (GreenSock) + ScrollTrigger
- **Icons:** [Lucide React](https://lucide.dev/)

## 🛠️ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/aliyudindudu/greenfold.git
   cd greenfold
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open the browser:**
   Navigate to [http://localhost:3000](http://localhost:3000).

## 🎨 Design System

- **Primary Color:** Cardboard Brown (#C4A484)
- **Secondary Color:** Dark Espresso (#3E2723)
- **Accent Color:** Fresh Leaf Green (#6B8E23)
- **Typography:**
  - Headings: *Saira Stencil One* (Stencil style for shipping/crates)
